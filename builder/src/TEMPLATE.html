Details

  • Clicking the number buttons changes the text color in the mock terminal windows. Clicking the color squares changes the background
  • The specific colors are made up from the 256 available by default in terminals
  • Changing either the text or background color updates the first two lines of the mock terminals to provide the codes for the colors
  • The number on the buttons and the first number in the mock terminal output lines is the 0-255 id for the color in the terminal. It's what you can use if you want to change stuff in your terminal via dotfiles
  • If you're wondering how useful this is given that themes and terminals with zillions of colors are a thing, the answer is "not particually, until it is". For example...
  • This entire thing came about because I'm trying tmux and wanted to change the default bright green of the status bar at the bottom. I found out how to do that from this post. And, you guessed it, you need an id for one of the 256
  • And, yes. I really built this just to pick colors for the tmux status bar
